# Log-in with Google

### Setting
Port: `./src/bootstrap/environments/port.ts`  
Firebase config: `./src/bootstrap/environments/firebase.ts`

### Development
```
npm run serve
```

### Deploy
```
npm run build
```

Firebase project name: `./.firebaserc`
```
firebase deploy
```
How to use firebase:
https://firebase.google.com/docs/hosting/quickstart
